A clean, modern, and persistent Pomodoro Timer application built with Vanilla JavaScript. Designed to help you focus on any task you are working on, such as study, writing, or coding.

## Features
- **Custom Timer Logic**: Default 25/5/15 minute intervals for Work, Short Break, and Long Break.
- **Persistence**: Your session history and timer state are saved automatically to `localStorage`. Reload the page without losing progress.
- **Dark / Light Theme**: Seamlessly switch between themes. Your preference is saved.
- **Responsive Design**: Works perfectly on Desktop, Tablet, and Mobile devices.
- **Audio & Visual Feedback**: Progress ring animation, completion sound, and visual shake effect.

## Technologies
- **HTML5**: Semantic markup.
- **CSS3**: Custom properties, Flexbox, Grid, and Animations. (No frameworks)
- **Vanilla JavaScript**: state management, local storage, and DOM manipulation.
- **GitHub Pages**: Static hosting.
